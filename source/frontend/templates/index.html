About

t-SNE: Assisted Parameter optimization by Approximating neighbourhood Similarity
Tool for hyperparameter optimization for dimensionality reduction with t-SNE in the context of word embeddings.
To start an optimization run, conduct the following two steps:
  • Upload dataset for uploading a new word vector file and
  • Create new run in order to set options for a new run (initial hyperparameter settings etc.).
If an already existing optimization run is to be continued, these steps are not necessary - in this case skip forward to Start/continue run.
